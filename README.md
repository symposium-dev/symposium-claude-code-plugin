# Symposium Claude Code Plugin

[Symposium][] plugin for Claude Code. Installing this plugin will instruct your agent to look for applicable skills for the crates you are using. See the [Symposium repository](github.com/symposium-dev/symposium) for more information.

*Be aware:* Using this plugin and skill will download the pre-compiled Symposium binary from the published artifcats of the [Symposium repository][].

[Symposium]: https://symposium.dev
